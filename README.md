Eco-Ride Urban Mobility & Fleet Management System is a Python-based application that simulates the management of electric vehicles across multiple city hubs. 
The project is designed on Object-Oriented Programming (OOP) principles, including encapsulation, inheritance, abstraction, and polymorphism.
The system supports electric cars and scooters it provides features such as searching, sorting, categorization.
Eco-Ride supports data persistence using CSV and JSON to save and restore fleet information, with each use case developed in a separate Git branch
